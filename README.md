## Overview

This repository contains my **Week 1 submission for the Cyberster Blue Team Internship**, where I built a small **SOC lab environment** and explored **SIEM operations using Wazuh**.

The objective was to understand how **endpoint logs travel to a SIEM** and how **security analysts review alerts and events**.

---

## Lab Environment

Three virtual machines were deployed using **VirtualBox**:

| Machine | Role |
|--------|------|
| Kali Linux | Testing / attack simulation |
| Ubuntu Server | Wazuh SIEM Server |
| Windows 10 | Monitored endpoint |

### Network Configuration

The lab used two network types:

- **Host-Only Network** → Communication between VMs  
- **NAT Network** → Internet access for updates and downloads
